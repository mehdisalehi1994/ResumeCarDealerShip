
پروژه چهارم مربوط به خودرو است

معرفی پروژه – وب‌سایت فروشگاه خودرو (Car Dealership)

این پروژه یک وب‌سایت استاتیک برای نمایش خودروهای لوکس و خدمات مرتبط با فروش و نگهداری خودرو است که به‌عنوان یکی از پروژه‌های پیشرفته‌تر مسیر فرانت‌اند من توسعه یافته است.

هدف اصلی پروژه، شبیه‌سازی یک وب‌سایت حرفه‌ای با تمرکز بر UI/UX، تجربه کاربری و تعاملات بصری پیشرفته بوده است.

ویژگی‌ها و پیاده‌سازی فنی

لودینگ صفحه (Preloader)
تجربه بارگذاری اولیه حرفه‌ای برای کاربران، به گونه‌ای که هنگام آماده‌سازی عناصر و ویدئوهای پس‌زمینه، تجربه نرم و حرفه‌ای ایجاد شود.

منوی برگری (Hamburger Menu)
برای دستگاه‌های کوچک، منوی ناوبری به صورت برگری درآمده و دسترسی آسان به بخش‌های سایت را فراهم می‌کند، همراه با انیمیشن باز و بسته شدن منو.

بک‌گراند ویدئویی و تصاویر تولیدشده با پرامپت (Prompt-Engineered Backgrounds)
ویدئو و تصاویر پس‌زمینه حرفه‌ای، با کیفیت بالا و با تکنیک Prompt Engineering طراحی شده‌اند که حس واقع‌گرایانه و جذابیت بصری بالایی ایجاد می‌کند.

ریسپانسیو حرفه‌ای (Mobile, Tablet, Desktop)

تغییر چیدمان بخش‌ها

تغییر اندازه فونت‌ها و فاصله‌ها

حفظ تجربه کاربری یکسان در همه دستگاه‌ها

استفاده از Bootstrap 4
تمام CSSهای موجود با Bootstrap 4 طراحی و پیاده‌سازی شده‌اند تا ساختار، Grid، فرم‌ها، ناوبری و دکمه‌ها به‌صورت استاندارد و حرفه‌ای پیاده‌سازی شود.
